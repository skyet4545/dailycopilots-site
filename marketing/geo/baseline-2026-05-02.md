# Bible Copilot — GEO Baseline Report

**Date:** 2026-05-02
**Tester:** Claude (automated, agent-mode)
**Scope:** 10 keyword queries × 5 engines = 50 (query × engine) cells
**Source of truth:** `baseline-2026-05-02.csv` in this folder. This document summarizes findings.

## Methodology

Ten queries were submitted to five engines. The CSV is the row-level record; this file gives the human-readable read-out.

**Engines that returned usable data:**
- **WebSearch (Google proxy)** — Anthropic's WebSearch tool. Returned Google-search-style top-10 links and snippets for all 10 queries. Used as a proxy for Google AI Overviews + organic SERP.

**Engines that could NOT be programmatically queried (gaps documented):**
- **Perplexity** — `https://www.perplexity.ai/search?q=…` returned a redirect/abort. No HTML body. Real Perplexity answers stream in via a websocket after JS hydration that requires a browser.
- **Bing / Copilot** — `https://www.bing.com/search?q=…&FORM=CHATIN` returned a 691 KB JS shell. Copilot's chat answer is rendered client-side from streaming responses; not present in the static HTML. No competitor URLs visible.
- **You.com** — `https://you.com/search?q=…` returned a 67 KB JS shell, no parseable citations. The `/api/youchatStreaming` endpoint timed out.
- **ChatGPT, Gemini, Meta AI** — no public HTTP search endpoints. All require authenticated, JS-rendered sessions inside their respective UIs. Manual test script provided below.

For Perplexity / Bing / You.com I sampled one query (`best AI Bible study app`) to confirm the shell-only behavior, then marked the remaining 9 queries `NOT_TESTED` rather than fabricate cells. **Carlos: please run those manually with the script below and overwrite the `NOT_TESTED` rows.**

## Manual test script — ChatGPT, Gemini, Meta AI, Perplexity, Bing/Copilot, You.com

Carlos: copy-paste each prompt verbatim into each engine. Prompts force the engine into list-mode so the citation footprint is visible. Record results back into the CSV's `NOT_TESTED` rows.

### Prompt template (copy this 10x, swap in each query)

> What are the best apps that do X for me? Please list 5–10 specific apps by name, give a short reason for each, and cite any sources you used. I'm on iOS.

…where X is each of the 10 queries, e.g. "AI Bible study with Greek and Hebrew analysis" or "inductive Bible study with Observe / Interpret / Apply".

### Recording template

For each (query, engine), record:
- `bible_copilot_cited`: YES / NO / INCONCLUSIVE
- `bible_copilot_url_if_cited`: the exact URL the engine hyperlinks (App Store listing, mybiblecopilot.com, netlify URL, or none if mentioned by name only — note `name-only`)
- `competitors_cited`: `;`-separated list of every named app
- `competitor_urls`: `;`-separated list of cited URLs in the same order
- `notes`: anything unusual (paywall, login wall, hallucinated app name, etc.)

### Engines and access

- **ChatGPT** — chat.openai.com → GPT-4o or GPT-5 with web search ON. Run each prompt in a fresh chat (no memory carryover). Note which model was used.
- **Gemini** — gemini.google.com → Gemini 2.x. Run each prompt in a fresh chat. Use the same Google account each time.
- **Meta AI** — meta.ai (web) or inside WhatsApp/Instagram. Run each prompt fresh.
- **Perplexity** — perplexity.ai → Default model with Web search. Note model used (Sonar / GPT-4 / Claude). Citations are explicit on Perplexity — capture them all.
- **Bing / Copilot** — bing.com/copilot or copilot.microsoft.com. Use "Balanced" or "Precise" mode; record which.
- **You.com** — you.com → Smart mode. Record which underlying model.

### Estimated time

10 queries × 6 engines × ~90 seconds per cell = ~90 minutes. Worth doing once at baseline. Re-run quarterly or after major site/content changes.

## What this baseline does NOT measure

- **App Store search** (separate algorithm; ASO not GEO).
- **Reddit / Discord conversations** where r/Bible, r/Christianity, r/Reformed mentions matter.
- **YouTube transcript surface** which Gemini and Google AI Overviews increasingly draw from.
- **Branded queries** ("Bible Copilot review", "is Bible Copilot legit") — these will return App Store + Netlify site by default and aren't useful baseline signal.

These should be added to a Phase-2 audit if the budget allows.
