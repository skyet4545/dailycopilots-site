# Scripture Copilot — Master Project

*AI-Powered Bible Study Companion*

---

## 🎯 Vision

Help Christians study the Bible deeply with AI that serves as a research assistant, not a replacement for Scripture. The Bible is the hero; AI is the copilot.

---

## 📊 Business Model

### Pricing Tiers

| Tier | Price | Features |
|------|-------|----------|
| **FREE** | $0 | 5 questions/day, 1 translation, basic study |
| **PRO** | $4.99/mo or $39.99/yr | Unlimited, all translations, journaling, reading plans |
| **FAMILY** | $9.99/mo or $79.99/yr | 6 accounts, shared notes, family devotions |

### Unit Economics

| Metric | Value |
|--------|-------|
| Cost per user/mo | ~$0.03 (API) |
| Revenue per PRO | $4.99/mo |
| Apple/Google cut | 30% |
| Net per PRO | $3.46/mo |
| Margin | 69% |
| Break-even | ~30 subscribers |

## 🛠️ Tech Stack

| Component | Tool | Why |
|-----------|------|-----|
| Framework | Expo (React Native) | Fast iteration, OTA updates, single codebase |
| Backend | Vercel Serverless | Free tier, easy deploy, scalable |
| AI | OpenAI GPT-4o-mini | Cost-effective, quality answers |
| Bible API | bible-api.com | Free, multiple translations |
| Payments | RevenueCat | Free to $2.5K MRR, handles both stores |
| Analytics | PostHog / Mixpanel | Free tiers, user behavior |
| Email | Loops / Buttondown | Free tiers for launch |

## 🎯 Differentiation

### vs. Generic AI (ChatGPT, Claude)
- **Theological guardrails** — Reformed, Scripture-first methodology
- **Structured study** — Observation → Interpretation → Theology → Application
- **Bible integration** — Seamless passage loading, cross-references
- **Mobile-first** — Designed for daily devotion use case

### vs. Existing Bible Apps (YouVersion, Logos)
- **AI-powered** — Ask any question about any passage
- **Simpler** — Not overwhelming with features
- **Affordable** — Not $500/year like Logos
- **Modern UX** — Clean, distraction-free
